# Rawn Snippets
This **package** contains some useful snippets for HTML, CSS, JavaScript, CoffeeScript, Sass &  Php.

## Install
```bash
 apm install rawn-snippets
```

- Html Comment

#### CoffeeScript

- Snippet Template
